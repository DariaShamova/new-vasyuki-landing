# Тестовый лендинг: New Васюки

Проект выполнен как тестовое задание по верстке для Яндекс Крауд.
Макет предоставлен в Figma.

**Используемые технологии:**
* BEM
* Adaptive layout
* Flexbox, grid
* Vanilla javascript slider and carousel
* Keyframes animation
